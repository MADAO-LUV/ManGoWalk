## 对于ManGoWalk                               
<a name="project-introduce"></a>  
*本项目因关注市面上的盲人手杖常常缺乏智能化，而且价格昂贵，功能不齐全等项目落脚点出发，经过多方综合考虑，打造出全新的智能盲杖---同时也适用于广大老年人群体*
*本项目app聚焦于盲杖寻路，为盲人提供电子导盲犬，更好地帮助盲人去到想去的地方。*
